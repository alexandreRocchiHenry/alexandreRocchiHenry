Projet Airbus - Détection de Changements Environnementaux

Projet académique - Computer Vision & Telecom Paris

Description du projet

Ce projet de groupe a pour objectif de détecter de manière non supervisée des changements atypiques dans des séries temporelles d’images satellitaires. La solution développée permet de hiérarchiser les anomalies environnementales afin de focaliser l’analyse sur des zones présentant des évolutions significatives de l’occupation du sol.

Modèle & Approche

Basée sur des modèles de fondation pour l’extraction d’embeddings locaux, l’approche compare ces représentations à l’aide de mesures de similarité (cosinus et euclidienne). Le pipeline intègre également des étapes de prétraitement, de clustering et de visualisation pour une analyse fine et quantitative des changements.

Prétraitement & Alignement

Implémentation d’algorithmes pour vérifier et corriger l’alignement des images, garantissant une base de données cohérente.

Extraction d’Embeddings

Utilisation de modèles de fondation pour générer des représentations locales, essentielles pour l’analyse fine des variations.

Mesures de Similarité

Application de distances cosinus et euclidiennes pour quantifier les changements significatifs entre les images.

Masques de Changement

Génération de masques basés sur la segmentation sémantique pour visualiser et quantifier les transitions locales.

Clustering & Hiérarchisation

Mise en place d’un pipeline de clustering (ex. avec Bertopic) pour classer et prioriser les anomalies détectées.

Analyse Statistique

Développement d’outils de visualisation pour suivre l’évolution des classes d’occupation du sol et évaluer la performance du modèle.

Projet de Vision par Ordinateur

Projet académique - Computer Vision

Description du projet

Ce projet vise la segmentation d'images satellitaires (4 bandes) en 7 classes. Les données sont issues du dataset DynamicEarthNet. L’objectif est de comparer les performances par continent, en entrainant le modèle sur quatres continents et en testant les résultats sur le cinquième.

Modèle

Le modèle utilise FarSeg (TorchGeo) avec ResNet comme backbone. Ceci nous permet d'avoir de bon résultats avec un mIou depassant les 0.7 sur un ensemble de train contenant tout les continents

Chargement des données

Implémentation d'un dataloader permettant de gérer les images RBG+NIR et répartition selon les coordonées gps dans les jeux de test ou de validation

Data parallèlisation et AMP

Mise en place de parallèlisation GPU et de GradScaler pour optimiser les temps d'entrainement. MixedPrecision pour optimisé la mémoire.

FarSeg & ResNet

L’intégration de FarSeg avec un ResNet pré-entraîné permet une segmentation plus robuste sur plusieurs types de terrains.

Stratégie d'entrainant

Defreezing progressif du modèle pour éviter la divergence et permettre d'affiner le réseau de neuronnes

Évaluation

Les métriques (IoU, précision, etc.) sont calculées pour analyser les performances et comparer les résultats selon les ensembles continentaux.

Projet - Classification d’Images

Projet personnel - Computer Vision

Description du projet

Ce projet personnel vise à classifier automatiquement différentes espèces (ici des reptiles) à partir d’images. L’objectif est d’implémenter un pipeline complet allant de la préparation des données (split, labellisation) jusqu’à l’évaluation finale par K-fold cross-validation.

Modèle

Un réseau CNN de type EfficientNet (B0 ou B5), pré-entraîné sur ImageNet, combiné à un ensemble d’augmentations pour améliorer la robustesse du modèle.

Pipeline de Données

Utilisation d’une architecture flexible pour charger, prétraiter rapidement des images et entrainer un modèle

Data Augmentation

Applications de transformations avancées (flip, rotation, zoom, etc.).

Approche de Transfer Learning

Fine-tuning d’EfficientNet pour l’identification multi-classes.

Cross-validation Stratifiée

Implémentation de K-Folds pour une meilleure généralisation, clé dans l’évaluation robuste de modèles de détection appliqués aux données satellitaires.

TensorFlow

Utilisation de Tensorflow. Utilisation de la “mixed precision” afin d’accélérer le temps d’entraînement sur de larges volumes d’images.

Évaluation & Metrics

Suivi via matrice de confusion et classification report pour valider la performance.

Projet - Analyse de Manifestos & Chartes

Projet académique - NLP

Description du projet

Ce projet se concentre sur l'analyse et le clustering de manifestos et chartes relatifs à l'IA et à l'éthique. L'objectif est d'extraire les thématiques principales et de regrouper les documents pour dégager les tendances et divergences dans les discours.

Modèle & Méthodologie

Implémentation d'une pipeline complète intégrant le prétraitement, la vectorisation et le clustering. Des techniques telles que BERTopic, LDA et GPTopic sont utilisées pour extraire et interpréter les thèmes sous-jacents.

Pipeline de Clustering

Mise en place d'une pipeline robuste pour prétraiter, vectoriser et regrouper les textes.

Extraction de Thèmes

Utilisation de BERTopic pour identifier des thématiques cohérentes et pertinentes.

Prétraitement des Textes

Nettoyage, tokenisation et normalisation des données pour garantir une analyse de qualité.

Évaluation des Clusters

Application de métriques de cohérence pour valider la pertinence des regroupements.

Transférabilité des Méthodes

Techniques applicables à d'autres domaines, notamment à la vision par ordinateur pour l'analyse d'images environnementales.

Visualisation Interactive

Création de visualisations interactives pour interpréter et communiquer efficacement les résultats.
